Protect Your Data

About the Game

48 hour game jam submission following the theme "Real world issues." This game is a roguelike shooter where the objective is to combat viruses infecting your computer. There are 7 different unlockable upgrades, 2 different bosses, and a developed story line. This project was developed using C#11 and the Unity Game Engine.

Controls

  • Space: Skip Dialog
  • WASD: Move character
  • Left Click: Fire Projectile
